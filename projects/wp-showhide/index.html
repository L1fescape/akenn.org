Hiding spoilers cannot be done easily in Wordpress.

Cue Wordpress Plugins.

The Wordpress plugins currently available either don't allow custom spoiler text or don't allow you to set the initial state of the spoiler content.

Cue Better WP ShowHide.

It looks something like this:

{% highlight html %} ["{Show Text}", "{Hide Text}", "{visible}"]
{Content goes here}
{% endhighlight %}

Configuration is pretty simple. Replace {myId} with a custom name to identify that individual spoiler in both the onclick attribute of the a tag and in the id attribute of the div tag.

Then set what you'd like the text spoiler link content to be when it's being shown by replacing {Show Text} with a custom value. Same goes for {Hide Text}.

Set the initial state of the spoiler, either hidden or visible. Do this by replacing {visible} with either of those values.

Finally set insert the content you want to hide by replacing {Content goes here} with your custom content. This can be anything from a YouTube video to 29,467 pictures of cats.

{% highlight html %} ["Show Text", "Hide Text", "visible"]
Snape kills Dumbledore.
{% endhighlight %}

Here's a fully working example using the code just above this sentence:

["View Spoiler", "Hide Spoiler", "hidden"]

Snape kills Dumbledore.

Hopefully that helps!
